# Reconocimiento_lengua_de_signos
Repositorio que contiene el código realizado para el TFG de Reconocimiento de lengua de signos.
Autora: Lucía Olmos Martínez
Tutor: Ginés García Mateos

## Ejecutar video.py
Para poder ejecutar *video.py*, en una terminal, ubicada dentro de la carpeta donde se encuentre el código, realizar el siguiente comando:
```
python video.py
```
Es necesario que esté presente en esa misma carpeta el archivo *hand_landmarker.task*.

## Ejecutar modelo.py
Para poder ejecutar *modelo.py*, en una terminal, ubicada dentro de la carpeta donde se encuentre el código, realizar el siguiente comando:
```
python modelo.py
```
Es necesario que los modelos entrenados estén presentes en la carpeta *modelos*.
